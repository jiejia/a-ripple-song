# A Ripple Song Podcast

A Ripple Song Podcast は、WordPress サイトでポッドキャストの公開と管理を行い、主要なポッドキャストプラットフォーム向けの Podcast RSS を生成できる WordPress プラグインです。

<img width="" src="../screenshot.png"/>

## 特長

- ポッドキャスト用の投稿タイプと分類を提供し、番組コンテンツの追加と管理をサポート
- 音声メタデータ（再生時間、ファイルサイズ、形式）を自動取得し、iTunes と Podcasting 2.0 の仕様に準拠
- RSS フィード生成を標準搭載し、主要なポッドキャストプラットフォームへ簡単に配信可能
- 多言語対応

## 技術スタック

- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress 用カスタムフィールドフレームワーク
- [getID3](https://github.com/JamesHeinrich/getID3) - 音声メタデータ解析ライブラリ
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP 名前空間分離ツール
