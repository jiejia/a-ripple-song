# A Ripple Song

A Ripple Song एक WordPress पॉडकास्ट प्लगइन है, जो WordPress में पॉडकास्ट प्रकाशित करने और प्रबंधित करने की सुविधा जोड़ता है, और प्रमुख पॉडकास्ट प्लेटफ़ॉर्म से सिंक करने के लिए Podcast RSS बना सकता है।

<img width="" src="../screenshot.png"/>

## दस्तावेज़

- [https://aripplesong.me/](https://aripplesong.me/)

## विशेषताएं

- पॉडकास्ट प्रकाशित और प्रबंधित करने के लिए पॉडकास्ट पोस्ट टाइप और कैटेगरी
- ऑडियो मेटाडेटा (अवधि, फ़ाइल आकार, फ़ॉर्मेट) का स्वतः निष्कर्षण, iTunes और Podcasting 2.0 मानकों के अनुरूप
- प्रमुख पॉडकास्ट प्लेटफ़ॉर्म पर एक-क्लिक वितरण के लिए बिल्ट-इन RSS जनरेशन
- अंतरराष्ट्रीयकरण समर्थन

## तकनीकी स्टैक

- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress कस्टम फील्ड फ्रेमवर्क
- [getID3](https://github.com/JamesHeinrich/getID3) - ऑडियो मेटाडेटा विश्लेषण लाइब्रेरी
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP नेमस्पेस आइसोलेशन टूल
