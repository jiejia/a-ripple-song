An all-in-one podcast theme for WordPress with a modern audio player and flexible widgets.

== Description ==

A Ripple Song is a podcast-focused theme for WordPress. It includes dedicated templates for episodes and posts, theme options, and a widget system for common content modules.

== Installation ==

1. Upload the `a-ripple-song` folder to `/wp-content/themes/`.
2. Activate the theme in Appearance â†’ Themes.
3. Configure widgets and theme options as needed.

== Frequently Asked Questions ==

= Does this theme register custom post types or taxonomies? =

No. Podcast post types/taxonomies should be provided by a companion plugin.
