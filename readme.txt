Classic WordPress theme with rich color schemes, Swup page transitions, widgets, and optional podcast features.

== Description ==

A Ripple Song is a classic WordPress theme. After installing the podcast plugin, you can use the audio player, podcast widgets, and related features.

Key features:

* 56 theme color schemes: 26 light themes and 26 dark themes
* Internationalization support
* Data tracking
* Swup-powered seamless page transitions
* Multiple widgets tailored to the theme style; some features require the podcast plugin
* Immersive audio player; requires the podcast plugin

Technical stack:

* Tailwind CSS
* daisyUI
* Alpine.js
* Lucide
* Simple Icons
* Swup
* Howler.js
* audioMotion-analyzer
